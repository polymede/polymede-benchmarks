Polymede Benchmarks

Base64Pad

This folder contains the benchmark results for the Base64Pad compiled with GHC 9.4.4 (aarch64), on the marvin reference node running macOS 13.2 (22D49).

Key Results

  • encodes 1,000,000 bytes in 457.2 μs using the ByteLazy pipeline
  • decodes to 1,000,000 bytes in 1.894 ms using the ByteLazy pipeline
